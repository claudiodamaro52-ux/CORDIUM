Início Suporte

Ajuda Rápida

Se você está com problemas, tente primeiro estas soluções:

  • Reinicie o programa
  • Verifique sua conexão com a internet
  • Atualize para a versão mais recente
  • Certifique-se de que o Google Chrome está instalado

Perguntas Frequentes (FAQ)

1. O programa não abre. O que fazer?

Verifique se o antivírus não bloqueou o arquivo. Adicione o programa à lista de permissões.

2. O ChromeDriver não foi encontrado.

O programa baixa automaticamente o ChromeDriver. Certifique-se de que o Google Chrome está instalado.

3. As imagens não estão sendo baixadas.

Verifique se a URL está correta e se a página realmente contém imagens.

4. O programa diz que há uma atualização, mas não instala.

Execute o arquivo Atualizador.exe manualmente na pasta do programa.

5. Como reinstalar o programa?

Baixe a versão mais recente na página de download e execute o instalador.

Requisitos do Sistema

  • Windows 10 ou 11
  • Google Chrome instalado
  • Conexão com a internet

Como Atualizar

  1. Abra o programa normalmente
  2. Se houver nova versão, o sistema avisará automaticamente
  3. Clique em “Sim” para atualizar
  4. O instalador será baixado e executado

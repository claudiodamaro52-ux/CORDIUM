Nosso Produto

Coletor de Imagens

O Coletor de Imagens é um software avançado que permite baixar imagens automaticamente do Google e Bing, com filtros de resolução, tipos, limite e muito mais.

Ele inclui:

  • Atualização automática
  • Instalador profissional
  • Histórico inteligente
  • Logs detalhados
  • Interface simples e intuitiva

DEVJSON

Manipulador de JSON online

Ferramenta web para manipulacao de arquivos JSON diretamente no navegador, sem instalacao. Normaliza, indenta, gera templates, identifica tipos e converte entre JSON e CSV.
